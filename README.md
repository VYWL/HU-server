# 스마트 환경에서의 이상행위와 사이버 공격에 대한 모니터링 및 대응 솔루션 제작 (Server)

스마트 환경 탐지 / 조치 및 모니터링 솔루션을 위한 백엔드 서버 👁

## 구성 언어 및 라이브러리와 런타임

-   Javascript
-   Express
-   Node.js

## 설치 방법

### 우선 git Clone

```sh
$ git clone https://github.com/VYWL/HU-server.git
$ cd HU-server
```

### 초기 패키지 설치

```sh
$ # 의존성 패키지들을 모두 다운받는 과정
$ npm install
```

### 로컬호스트 시작

```sh
$ npm start
```
